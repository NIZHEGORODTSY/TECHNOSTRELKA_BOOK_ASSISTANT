# BookSearch

BookSearch - сервис, который позволяет легко ориентироваться в текстах произведений мировой литературы

**Сайт доступен доступен по [ссылке](http://46.29.165.41:5000/)**

## Стэк технологий:
- Pinecone - для хранения книг и поиска фрагментов в них;
- DeepSeek - для ответов на интересующие пользователей вопросы;
- Python &#8594; Flask + Jinja2 - для веб-сайта

## Доступные возможности:
1. Поиск фрагмента книги по описанию
2. Точечные ответы на вопросы по книгам от ИИ-помощника
3. Добавление своих книг в общую библиотеку

## Как запустить сервис?

Для запуска сервиса убедитесь, что на устройстве установлен `git` и `python:3.14`

Клонируйте репозиторий себе на устройство: `git clone https://github.com/NIZHEGORODTSY/TECHNOSTRELKA_BOOK_ASSISTANT.git`

Установите все необходимые зависимости: `pip install -r requirements.txt`

Создайте переменные окружения `DEEPSEEK_API_KEY` и `PINECONE_API_KEY`, в которых будут храниться соответственно ключи к API Pinecone и Deepseek (предварительно их надо создать). Можете создать их напрямую на устройстве или же создать в папке проекта файл `_.env` следующего содержания:
```
DEEPSEEK_API_KEY="ваш-ключ"
PINECONE_API_KEY="ваш-ключ"
```

После этого запустите сервис: `python src/app.py`

Сайт будет доступен по адресу `localhost:5000`
